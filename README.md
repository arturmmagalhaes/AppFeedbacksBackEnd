# AppFeedbacksBackEnd

# Executar a aplicação com mysql na nuvem

``` 
  git clone https://github.com/arturmmagalhaes/AppFeedbacksBackEnd.git
  
  npm install
  
  npm start
```

# Executar a aplicação sem mysql na nuvem

  baixar o mysql
  
``` 
  git clone https://github.com/arturmmagalhaes/AppFeedbacksBackEnd.git
  
  npm install
  
  npm run migration
  
  npm start
```
